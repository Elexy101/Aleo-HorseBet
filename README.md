## Requirements
- Leo compiler (can be installed as an extension on VS-code)
- VS Code Editor
- snarkos setup
- leo/aleo setup

## Build Guide


To compile this Aleo program, run:
```bash
leo build
```

To execute this Aleo program, run:
```bash
leo run mint_public
leo run bid_public
leo run reward
```
